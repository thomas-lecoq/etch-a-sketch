# odin-etch-a-sketch
This is a beginner javascript project made via the Odin Project html and css course.
This exercise was the occasion to practice basic JS, especially DOM manipulation & event handling.

## Elements studied through this project :
- problem solving method
- basic DOM manipulation
- event handling
- functions declaration, arrow functions, arrow function expressions
- control flow
- global & local scope
- debugger

## Many mistakes made through this project, this serves as a reminder on element to consider for future project :
- commit & push made with Claude resulting in poorly trackable commits
- non atomic git commit were made
- consistency of naming in css class needs to improve
- code needs to get more modular with a higher number of functions
- javascript files organization needs to improve, aiming for a better separation of concerns and import / export flow between files
